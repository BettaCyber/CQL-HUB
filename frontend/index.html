How to Contribute:

Want to share your own query?

  • Use this form to create a new query.
  • Click Submit Query to send it directly for review, or Download YAML to submit it manually via a GitHub Pull Request.
  • After approval, your query will appear here on CQL Hub.
Contact information will not be shared publicly and is only used to reach out if we have questions.
Separate multiple MITRE IDs with commas

How to Contribute:

Want to share a lookup file?

  • Upload a CSV file that can be used with CQL's match() function.
  • Please make sure to name the file according to its' use case.
  • Click Submit Lookup File to send it for review.
  • After approval, the file will be available on CQL Hub for use in queries.
